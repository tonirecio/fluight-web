Fluight-Core

React UI Library for the Travel Industry

This project aims to create a ReactJS component library specifically tailored for building web interfaces within the travel industry.

Given the time constraints of the current assignment, and to provide a focused deliverable within the broader scope of the travel industry, this initial phase of the library will concentrate on the commercial flight domain.

The project name, FLUIGHT, is a convergence of the English words "Flight" and "UI" (User Interface), reflecting the core focus of this library.
